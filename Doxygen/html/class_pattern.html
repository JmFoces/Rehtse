Class that implements a Pattern. A pattern is composed by: BPF filter Regex Plus some replacement stuff that fits the operations for what this program is thought to be. Replacement is composed by a regex and a replacement string. Constructs a pattern from: bpf_string(bpf), regex(regex), replacement_regex(replacement_regex), raw_replacement(raw_replacement) It compiles the bpf program and regex. Raises: BPF_ERR if cannot compile the filter.
